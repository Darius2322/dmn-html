    🔧 Google Sign-In + Admin Setup

    Follow these steps once in your Supabase dashboard

    First: Run this SQL in your Supabase project → SQL Editor to create the required tables:

    -- 1. Portfolio table create table if not exists portfolio ( id uuid default gen_random_uuid() primary key, title text not null, description text, url text, image text, tags text, category text default 'business', featured boolean default false, sort_order int default 0, created_at timestamptz default now() ); alter table portfolio enable row level security; create policy "public_read_portfolio" on portfolio for select using (true); create policy "admin_all_portfolio" on portfolio for all using (auth.jwt() ->> 'email' = 'dmnsolutions63@gmail.com') with check (auth.jwt() ->> 'email' = 'dmnsolutions63@gmail.com'); -- 2. Feedback table (if not already created) create table if not exists feedback ( id uuid default gen_random_uuid() primary key, user_id uuid references auth.users, user_email text, user_name text, rating int check (rating between 1 and 5), service text, comment text, created_at timestamptz default now() ); alter table feedback enable row level security; create policy "public_read_feedback" on feedback for select using (true); create policy "auth_insert_feedback" on feedback for insert with check (auth.uid() = user_id); create policy "admin_delete_feedback" on feedback for delete using (auth.jwt() ->> 'email' = 'dmnsolutions63@gmail.com');
    Enable Google Sign-In
    1

    Go to Google Cloud Console

    Visit console.cloud.google.com → Create a new project (or use existing).

    2

    Enable OAuth 2.0 Credentials

    Go to APIs & Services → Credentials → Create Credentials → OAuth client ID. Choose Web application.

    3

    Add Authorized Redirect URI

    In the OAuth client settings add this exact URL as an Authorized Redirect URI:

    https://eogezhtvaoabtvsiidth.supabase.co/auth/v1/callback
    4

    Copy Client ID + Secret

    After creating, copy the Client ID and Client Secret.

    5

    Add to Supabase

    In your Supabase dashboard → Authentication → Providers → Google. Enable it and paste the Client ID and Secret. Save.

    6

    Add Site URL to Supabase

    In Supabase → Authentication → URL Configuration, add your website URL to Redirect URLs.

    https://your-domain.com
